# req.subdomains
An array of all the subdomains in this request's URL.

### Usage
```javascript
req.subdomains;
```

### Example

If the requested URL was "https://ww3.staging.ibm.com":

```javascript
req.subdomains;
// -> ['ww3', 'staging']
```
